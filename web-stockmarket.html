Stock market application

Goal

This web application uses an API provided by marketstack.com. This API allows you to query

  • stock exchanges
  • currencies
  • tickers (in USA)
  • tickers (worldwide via marketstack.com)
  • EOD prices for an entered ticker
  • EOD prices for Sofina

Techniques used

The following techniques were used:

  • flask server + jinja templates
  • REST services (no storage of data)
  • matplotlib for chart
  • generate an image in the server on the fly, pass a url to a jinja template and foresee a route to get the picture
  • Python library for api: requests
  • Python library for the web server: flask
  • Python libraries for hiding the API key: dotenv.main, os
  • Python libraries for generating a chart with EOD stock prices: matplotlib, numpy
  • Documentation of the API provider
