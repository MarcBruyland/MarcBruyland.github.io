Flight deals

Goal

The goal of this application is to obtain interesting flight deals. You manage a few parameters in a Google doc sheet that is exposed via the site sheety. You search for interesting prices according to these parameters. You get alerts via mail or via sms.


Techniques used

The following techniques were used:

  • Google doc sheet exposed via an api through sheety.
  • The tequila endpoint gives you access to flight information.
  • With a notification manager, messages can be sent either via Twilio (sms) or via mail.
  • A .env file hides all sensitive info. Access via Python library dotenv.
