Todo manager

Goal

This web application lets users manage their todo list.


Techniques used

The following techniques were used:

  • CRUD application: create/read/update/delete todos
  • SQLite database: Python initialization script, DBBrowser for SQLite
  • SQLite database: table todo, table user
  • Flask server written in Python
  • Jinja2 templates
  • CSS styles including bootstrap styles
  • Python libraries: flask, flask_sqlalchemy, flask_bootstrap, wtf forms
  • Security - Python libraries: bcrypt (hash password), os, binascii (generate token), datetime
