Scrape table of the Premier League

Goal

Create a dictionary with the information of the table of the Premier League from an Internet web page.


Techniques used

The following techniques were used:

  • the Python library BeautifulSoup for pulling data out of HTML and XML files
  • the Python library Requests, a simple, yet elegant, HTTP library
  • the Python library json, for storing a Pyhon dictionary
