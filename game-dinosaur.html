Dinosaur game

Goal

The goal of this project is to automate the playing of the dinosaur game which can be found at this location.


Techniques used

The following techniques were used:

  • PyAutoGUI lets your Python scripts control the mouse and keyboard to automate interactions with other applications.
  • Python library opencv-python is used for displaying an image.
  • The program repeatedly takes screenshots and determines variations in color which trigger the dinosaur to jump over obstacles that are coming its way.
