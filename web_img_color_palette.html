Image Color Palette Generator

Goal

This web application lets you find the most common colors in an uploaded image.


Techniques used

The following techniques were used:

  • Flask server
  • Jinja2 templates
  • numpy for handling the pixels in an image
  • Python Imaging Library (PIL)
  • sorting a dictionary on values
  • inverting the list order
  • HTML input field of type "file"
