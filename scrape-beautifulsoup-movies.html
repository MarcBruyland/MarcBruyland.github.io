Scrape top100 movies

Goal

Starting from a webpage on the Internet, this application creates a file with the names of the 100 most popular movies in history.


Techniques used

The following techniques were used:

  • the Python library BeautifulSoup for pulling data out of HTML and XML files
  • the Python library Requests, a simple, yet elegant, HTTP library
