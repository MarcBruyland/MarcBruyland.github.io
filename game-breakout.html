Breakout

Goal

The goal of this game is to tear down a wall and earn as much as possible points. Therefore you can use a paddle to bounce back a ball. A ball can be bounced back by a brick in the wall as well. Each brick disappears when hit; each brick is also worth a number of points (yellow=1, green=3, orange=5, red=7).


Techniques used

The following techniques were used:

  • Python scripts (with a GUI interface based on the Turtle library)
  • Object-oriented programming (classes "wall", "ball", "paddle", "scoreboard")
