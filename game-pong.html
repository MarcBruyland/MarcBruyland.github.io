Pong

Goal

Two players can play a game of pingpong. The paddles are moved with up an down arrows and with 's' and 'w' keys.


Techniques used

The following techniques were used:

  • Python scripts (with a GUI interface based on the Turtle library)
  • Object-oriented programming (classes "Ball", "Paddle", "Scoreboard")
