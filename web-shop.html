Web shop

Goal

This web shop application lets you buy toys. It includes the payment handling with the payment platform STRIPE.


Techniques used

The following techniques were used:

  • SQLite database with a db server that exposes REST services
  • Flask webshop server
  • Jinja2 templates
  • hiding sensitive info in a .env file
  • using the STRIPE platform for payment handling
  • 'requests' for sollicitating the REST services
  • 'jsonify' is used in the db server for returning data coming from the db
  • Python libraries: flask, flask_sqlalchemy, flask_bootstrap, forms, sqlite3, bcrypt (hash password), binascii (token generation), datetime, os, smtplib, stripe, uuid, requests, json
