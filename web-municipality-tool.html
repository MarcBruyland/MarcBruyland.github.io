The Municipality Tool

Goal

With this application the municipalities in Belgium will be able to fix mapping issues that occur when migrating from their own non standardized address data to new standard BEST addresses.


Techniques used

The following techniques were used:

  • React front-end (multi-language: nl,fr,de)
  • Express Node.js backend server
  • Mongo db database
  • APIs: fetching data from the db, updating the db, downloading files (log, todo, result)
  • Security: user db, username/password, token
  • Python scripts: for batch processes (daily statistics + todo files, weekly result files + new issue files)
  • Docker images and containers
  • Documentation: user manuals, (public) instruction videos
